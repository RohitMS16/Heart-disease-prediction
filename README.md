Heart Disease Prediction using Machine Learning
Project Overview

This project aims to predict whether a person is likely to have heart disease based on various medical attributes such as age, sex, chest pain type, blood pressure, cholesterol level, and other health indicators. The goal of this project is to build a machine learning model that can assist in early detection of heart disease, which can help in medical decision-making.

This project demonstrates the complete machine learning workflow including data preprocessing, exploratory data analysis, model building, and model evaluation.

Dataset Information

The dataset used in this project is the Heart Disease dataset, which contains medical information about patients. The dataset includes features such as:

Age
Sex
Chest Pain Type
Resting Blood Pressure
Cholesterol
Fasting Blood Sugar
Resting ECG
Maximum Heart Rate
Exercise Induced Angina
ST Depression
Slope of ST Segment
Number of Major Vessels
Thalassemia
Target (Heart Disease: Yes/No)

The target variable indicates whether the patient has heart disease or not.

Model Performance



The model achieved an accuracy of approximately 77%. The confusion matrix and classification report show that the model performs reasonably well in predicting heart disease. However, there is still room for improvement, and the model could be enhanced using hyperparameter tuning, feature engineering, and more advanced algorithms such as Random Forest or Gradient Boosting.
